# A Design of Power Supply Systems — Minimal Eleventy Site Template

This repository is a ready-to-deploy Eleventy site (digital-garden style) with a dark theme,
cards layout and tag support — inspired by your example site.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Deploy to Vercel:
- Create a new GitHub repository and push these files.
- In Vercel, import the repo, build command `npm run build`, output directory `_site`.
